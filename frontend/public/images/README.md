# Farmer Logo Image

Please place your farmer logo image here with the filename: `farmer-default-logo.png`

Supported formats: PNG, JPG, JPEG, SVG, WebP

Recommended size: 128x128 pixels or larger (will be scaled to 48x48px in the UI)

The image should be a portrait of a farmer that will be used as the default logo when a farmer's photo is not available from the FFA API.
